# div & span
## div 
块及元素 block
display:block
默认自成一块区域，独占一行，可设置高度
## span 用来组合行内元素
行内元素 inline 
display:inline
默认：相当于文字，从左到右，组成一行，不可设置高度
## img行内块元素
 行内块元素 inline-block
 display:inline-block/inlien
 与行内元素显示规则一致，但可以设置高度
 一般，用来排版的元素是块级元素
 一般，用来修饰文字的元素，是行内元素
 块级元素和行内元素可以相互转化，通过display属性。
